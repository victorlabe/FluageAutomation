Versions du logiciel

Téléchargez ici les différentes versions de FluageAutomation à installer sur les postes.

Le logiciel peut être installé sur une clé USB pour une utilisation portable, ou directement sur le disque dur du poste.

Compatibilité

Le logiciel est compatible avec les systèmes Windows uniquement. Assurez-vous de choisir la version adaptée à votre poste et de suivre les instructions d'installation fournies.
Légende des versions :
Vert Version recommandée et à utiliser.
Orange Version ancienne mais encore utilisable avec quelques modifications.
Rouge Version fortement déconseillée.
